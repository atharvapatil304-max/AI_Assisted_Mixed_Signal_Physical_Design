# Power Distribution Network (PDN) Hook

## Objective

Connect the analog hard macro power pins to the top-level power grid generated by OpenLane.

---

## Generated Hook

```text
FP_PDN_MACRO_HOOKS = "AMUX2_3V VPWR VGND VPWR VGND"
```

---

## Explanation

| Field | Description |
|-------|-------------|
| AMUX2_3V | Macro instance |
| VPWR | Macro power pin |
| VGND | Macro ground pin |
| VPWR | Top-level power net |
| VGND | Top-level ground net |

---

## Purpose

This hook ensures that:

- The macro is connected to the global power grid.
- OpenLane correctly generates the PDN.
- LVS connectivity is maintained.
- Power routing includes the analog macro.

---

## Validation

- Verified against macro pin names.
- Compatible with OpenLane mixed-signal integration.
