# Prompt 05 - Macro Pin-Access Debugging

## Prompt

An analog hard macro is failing the pin-access stage during routing.

Analyze the issue by:

1. Identifying likely root causes.
2. Verifying LEF pin definitions.
3. Inspecting pin geometry and routing layers.
4. Evaluating routing accessibility.
5. Reviewing obstruction (`OBS`) definitions.
6. Recommending changes to improve routability.
7. Outlining a systematic debugging procedure.

Return only:

* Root-cause checklist
* LEF verification checklist
* Step-by-step debugging workflow
* Recommended fixes
