# Prompt 01 – Repository Analysis

## Prompt

Analyze the provided repository and generate a technical summary of its structure and implementation.

### Tasks

1. Identify the top-level design module.
2. List all analog macro files and describe their purpose.
3. Identify the RTL source files.
4. Locate the LEF, LIB, GDS, DEF, and Verilog views associated with the analog macro.
5. Identify the OpenLane configuration files and explain their roles.
6. Describe the complete implementation flow from RTL to GDSII.
7. Explain the relationship between the digital design, analog macro, and physical design files.
8. Generate a dependency graph illustrating how the design files interact throughout the implementation flow.

### Output

- Repository hierarchy
- Design file summary
- Dependency graph
- RTL-to-GDSII implementation flow
- Key observations
