# Demonstration Video

**🎬 Demonstration Video:** *(https://drive.google.com/file/d/134JXLbk1jWJxCRZV0ymYh94aXrRBSrWa/view?usp=sharing)*

A screen-recorded demonstration video (maximum **20 minutes**) is included as part of this project to show how the complete AI-assisted circuit design workflow can be reproduced on another system with the required tools and PDKs already installed.

## Video Contents

- **Project Overview**
  - Project introduction
  - Reference repository
  - Selected circuit block
  - Design objective
  - Expected results

- **AI-Assisted Design Workflow**
  - AI tool/model used (ChatGPT)
  - Prompts used during development
  - Generated circuit, netlist, and testbench files
  - Project folder structure
  - Commands executed
  - Simulation workflow
  - Errors encountered and debugging process
  - Fixes applied
  - Final verified simulation results

- **Verification**

  - Output verification
  - Final observations
  - Reproducibility of the complete workflow

The demonstration focuses on practical execution rather than theory, allowing another user to reproduce the complete design flow from the problem statement to the verified output.
