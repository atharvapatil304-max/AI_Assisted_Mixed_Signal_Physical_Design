# Truth Table

| SEL | Connected Input | Output |
|-----|-----------------|--------|
| 0 | A | X = A |
| 1 | B | X = B |

---

## Functional Description

The analog multiplexer consists of two CMOS transmission gates.

- When **SEL = 0**, the transmission gate connected to **Input A** is ON while the transmission gate connected to **Input B** is OFF.

- When **SEL = 1**, the transmission gate connected to **Input B** is ON while the transmission gate connected to **Input A** is OFF.

This configuration provides low-resistance, bidirectional analog switching with full rail-to-rail signal transmission.
